# VK Бот для разработчика

Этот бот работает только для разработчика с ID 532796366 и предназначен для управления двумя чатами.

## Функциональность

- Авторизация по ID разработчика
- Команда `/test` с ответом "Слушаюсь!!"
- Работа в двух чатах: Архив и Обычный чат

## Настройка

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Создание бота VK

1. Перейдите в [Управление сообществом VK](https://vk.com/groups?act=manage)
2. Создайте новое сообщество или выберите существующее
3. В настройках сообщества перейдите в "Разделы" → "Работа с API"
4. Создайте ключ доступа с правами:
   - Сообщения сообщества
   - Доступ в любое время

### 3. Настройка конфигурации

1. Скопируйте файл `.env.example` в `.env`:
```bash
copy .env.example .env
```

2. Заполните `.env` файл реальными значениями:
```
VK_BOT_TOKEN=ваш_токен_бота
VK_GROUP_ID=id_вашего_сообщества
```

### 4. Запуск бота

```bash
python bot.py
```

## Как получить ID чатов

1. Запустите бота
2. Напишите любое сообщение в чате где должен работать бот
3. Бот запишет ID чата в консоль
4. Запомните эти ID для настройки чатов

## Структура чатов

- **Архивный чат**: для команд другому боту
- **Обычный чат**: для команд текущему боту

## Команды

- `/test` - тестовая команда, ответ: "Слушаюсь!!"
